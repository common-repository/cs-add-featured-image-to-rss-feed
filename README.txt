== Description ==

This plugin will Add Featured Image or Post Thumbnail to RSS Feed.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'CS Add Featured Image to RSS Feed'
3. Activate CS Add Featured Image to RSS Feed from your Plugins page.

= From WordPress.org =

1. Download CS Add Featured Image to RSS Feed.
2. Upload the 'cs-add-featured-image-to-rss-feed' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate CS Add Featured Image to RSS Feed from your Plugins page.
